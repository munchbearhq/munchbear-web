# AI-Generated Code Policy

The use of AI-assisted coding tools to generate or autocomplete code is not
permitted.

## Why this policy exists

We believe that:

- Deep understanding comes from thinking, struggling, and solving.
- Over-reliance on AI tools can weaken core engineering skills.
- Small, human-made, reviewable changes tend to preserve quality and
  maintainability over the long run.

### What is allowed

- Reading documentation.
- Discussing ideas with other humans.
- Writing code entirely by yourself.
- Learning of concepts using AI Chatbots without indexing or giving the
  repository as context.

### What is not allowed

- Generating code using AI tools.
- Copy-pasting AI-generated snippets.
- Using AI autocomplete systems for implementation.
